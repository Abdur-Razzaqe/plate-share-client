1. Website Name: plate-share
   PlateShare is a full-stack MERN application that allows users to share surplus food with the community. User can post food items they want to donate, and other users can browse and request these food items.

2.Live site URL: https://plate-share-999.netlify.app/

3. Features:
   a. Authentication:-
   -- Firebase Email/Password login & Registration
   -- Google Sign-In
   -- Logout functionality

b. Food Management (CRUD):
--Add, update, delete food (private routes)
--Manage your own food posts
--View food details including donator info

c. Food Request system:
--Request available foods
--Food owner can accept/reject requests
--Automatic food status update (Available--Donated)

d. Search & Filters:
--Search by exact name
--Display featured foods dynamically

e. Responsive & Modern UI:
--Navbar & Footer with dynamic elements
--Banner, How It Works, Our Mission sections
--Framer motion animation on the homepage

4. Tech Stack:
   a. Frontend: React.js, React-Router, TailwindCSS, DaisyUI, Framer Motion,React-Hot-Toast, React-Icon, Sweet alert

b. Backend: Node.js, Express.js, MongoDB

c. Authentication: Firebase Authentication
d. Hosting: Netlify (Fronted), Vercel (Backend)
